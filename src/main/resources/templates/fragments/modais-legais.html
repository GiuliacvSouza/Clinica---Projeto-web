Termos de Serviço

Última atualização: Junho de 2026 · Clínica Dentária

Nota: Este documento foi elaborado exclusivamente para fins académicos no âmbito de um projeto universitário. O seu conteúdo é simulado e não constitui um instrumento jurídico vinculativo.

Estes Termos de Serviço regulam a utilização da plataforma digital da Clínica Dentária. Ao criar uma conta ou utilizar a plataforma, o utilizador declara que leu, compreendeu e aceita estes termos.

1. Objetivo da plataforma

A plataforma digital da Clínica Dentária tem como objetivo facilitar o acesso dos pacientes a serviços de saúde oral, permitindo o agendamento de consultas, a gestão de dados pessoais e a comunicação com a clínica.

2. Conta de utilizador

Para utilizar os serviços da plataforma, o utilizador deve:

  • Fornecer informações verdadeiras, completas e atualizadas no momento do registo.
  • Manter a confidencialidade das suas credenciais de acesso.
  • Não partilhar a sua conta com terceiros.
  • Informar a clínica caso suspeite de acesso indevido à sua conta.

3. Utilização adequada

O utilizador compromete-se a utilizar a plataforma de forma responsável, sendo proibido:

  • Introduzir dados falsos ou enganosos.
  • Tentar aceder a informações de outros utilizadores sem autorização.
  • Utilizar a plataforma para fins comerciais não autorizados.
  • Realizar ações que possam comprometer a segurança ou estabilidade do sistema.

4. Comunicações

Ao criar uma conta, o utilizador aceita receber comunicações relacionadas com a sua conta, incluindo confirmações, mensagens administrativas e recuperação de palavra-passe.

5. Limitação de responsabilidade

A Clínica Dentária não se responsabiliza por danos resultantes de interrupções temporárias do serviço ou falhas técnicas alheias ao seu controlo.

6. Alterações aos termos

A Clínica Dentária pode atualizar estes Termos sempre que necessário. A versão mais recente estará disponível para consulta na plataforma.

7. Nota académica

Este documento foi criado no contexto de um projeto académico. O seu conteúdo é meramente ilustrativo e não tem validade jurídica.

Política de Privacidade

Última atualização: Junho de 2026 · Clínica Dentária

Nota: Este documento foi elaborado exclusivamente para fins académicos. O seu conteúdo é simulado e não constitui um instrumento jurídico vinculativo. O tratamento de dados descrito segue princípios gerais do RGPD a título ilustrativo.

Esta Política de Privacidade explica, de forma simples, como a Clínica Dentária trata os dados pessoais fornecidos pelos utilizadores através da sua plataforma digital.

1. Responsável pelo tratamento de dados

O responsável pelo tratamento dos dados pessoais recolhidos através desta plataforma é a Clínica Dentária, no âmbito deste projeto académico simulado.

2. Dados pessoais recolhidos

No momento do registo, poderão ser recolhidos os seguintes dados:

  • Nome completo.
  • Endereço de e-mail.
  • Número de telemóvel, quando fornecido.
  • Palavra-passe (armazenada de forma encriptada).

3. Finalidade do tratamento

Os dados pessoais recolhidos são utilizados para:

  • Criar e gerir a conta do utilizador.
  • Permitir a autenticação na plataforma.
  • Enviar comunicações relacionadas com a conta.
  • Permitir a recuperação de palavra-passe.

4. Segurança dos dados

A Clínica Dentária adota medidas técnicas para proteger os dados pessoais contra acesso não autorizado, incluindo encriptação de palavras-passe e tokens de recuperação com validade limitada.

5. Partilha de dados com terceiros

Os dados pessoais dos utilizadores não são vendidos, alugados ou cedidos a terceiros para fins comerciais.

6. Direitos do utilizador

O utilizador pode solicitar o acesso, correção ou eliminação dos seus dados pessoais, de acordo com a legislação aplicável (RGPD).

7. Conservação dos dados

Os dados serão conservados apenas pelo período necessário para cumprir as finalidades para as quais foram recolhidos.

8. Nota académica
